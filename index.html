Community projects and Developer Tools

Oscilloscope Online

Online Serial Graph Plotter using p5.js: Useful for research work and live experimentation data. The graphs can help recognize patterns and other similarities. The user can use it directly without any downloading or installation required.

Graph Plotter

Library for Processing IDE: Plots a line graph according to user's code. The user has full access to the source code and can do almost anything including using logarithmic scales as well as factorial scales. As long as the user can code it, anything is possible.

Arduino Language Support

Extension for VS Code: Declares Arduino as a separate computer language in VS Code. This isolates it from C++ meaning that the user can now have 2 isolated environments making it easier to program.

Arduino Toolbox

Utility tool: For testing and debugging a Robotic Arm

It includes:

  • Home Page
  • Serial Monitor
  • Gyro-Reader
  • Servo Arm Controller

Attendance System

NodeMCU based attendance system using RFID. A reliable and secure method for assigning class attendance

The project uses 3 security keys

  • Certified Attendance markers
  • Admin
  • Student Registration

Self-Learning Projects for Computer Engineering



5 Bit Computer Memory

A 5 bit Memory unit made using raw electronics, it is equivalent to 5 boolean variables.

Automatic Night Light

A Night Light that turns ON and OFF automatically based on the lighting of the room it is present in. The project does not use an Arduino or Microcontroller of any kind.

Laser Security System

A Security System based on Laser Lights. Just like in the movies, when the laser is cut, an alarm sounds. The project doesn't use an Arduino and instead uses the 5 Bit Computer Memory, a previous project. It can also tell which Laser was broken.
